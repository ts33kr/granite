This middleware is a wrapper around the toobusy module providing the functinality that helps to prevent the server shutting down due to the excessive load. This is done via monitoring of the event loop polling and rating the loop lag time. If it's too big, the request will not be processed, but simply dropped. This is a config wrapper.
