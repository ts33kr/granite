This is the place where actual context sources minification and reduction takes place. Get the contents of the resources at the established path. It is a good idea for this HTTP method to be idempotent. As the rule, this method does not have to alter any contents or data of the resource. Use for unobtrusive retrieval of resources. The method is an HTTP verb, coherent with the REST interface.
