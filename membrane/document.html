Descriptor of some method of arbitrary service, in a structured and expected way, so that it can later be used to programmatically process such documentation and do with it whatever is necessary. This approach gives unique ability to build self documented APIs. So this class is basically a typed, programmatic data storage.

module.exports.Document = class Document extends Archetype
