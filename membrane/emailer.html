This is an ABC service intended to be used only as a compund. It provides the facilities to send emails from the services that mix this compound in. It draws the necessary configuration data, then sets up all the internal objects required. A mailer configuration is done once only, then mailer persists on the kernel and reused.

module.exports.EMailer = class EMailer extends Service
