This abstract compound provides the internal service API for the automatic generation of necessary links and then inclusion of the required tags to use the confgured fonts provided by the Google Fonts service that generates the CSS stylesheets with settings. Please refer to the Google docs to get info on the fonts format.

Add the described font to the font request that is going to be compiled and emited when the context is assembled. Description of a font is formed of a font family name and a vector of the typographic descriptions, such as sizes and styles. Values in the vector can be strings or any others - all are stringified.
