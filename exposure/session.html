This zombie service exposes session storage engine implementation. This implementation uses Redis to store the session data. It does make use of RedisClient and other wirings specific to framework. It is highly recommended to use this storage engine in production environments. Please refer to plumbs module for how to make use. Normally, you would not use this zombie class instance directly.
