This abstract compound provides the curious functionality that is greatly improving the way you would construct the client side code that is responsible for rendering UI elements. It allows you to run client side (external) code once the specified selector is available. All of this lets you easily implement the viewport/slow architecture. It also allows you to react on when selector changes or disappears.
